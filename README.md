# 🦖 Jurassic Explorer (Liquid Galaxy Project)

## Overview
Jurassic Explorer is an interactive visualization project built for the Liquid Galaxy platform. It allows users to explore dinosaurs, prehistoric environments, and geological events using Google Earth and KML.

## Features
- 🌍 Interactive Earth exploration
- 🦖 Dinosaur information system
- ⏳ Timeline-based navigation
- 🎥 Multi-screen Liquid Galaxy visualization
- 🔊 AI narration (planned)

## Tech Stack
- Flutter (Mobile App)
- KML (Visualization)
- Google Earth
- Liquid Galaxy

## Project Structure
- /kml → KML visualization files
- /app → Mobile app (Flutter)
- /assets → Images and media
- /docs → Proposal and documentation
